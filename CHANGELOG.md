# Changelog

All notable changes to the AutoSay extension will be documented in this file.

## [1.0.0] - 2026-02-07

### Added
- Initial release
- Customizable greeting message
- Multiple voice options (Samantha, Victoria, Karen, Moira, Tessa, Fiona)
- Configurable speech rate (90-720 WPM)
- Greeting on editor startup
- Greeting on new AI chat (`Cmd+Shift+L`)
- Debug command to show available chat commands
- Support for Cursor IDE
